This extension scans PDF files for metadata. These can contain information on usernames, personal names, software used, etc.

If this extension is loaded it will be run on passive scans.


Supported Metadata:
  • Document Information
  • XMP Metadata
Requirements:
  • Jython 2.7
  • python-chardet
  • python-pdfminer
  • xerces.jar (for XMP parse support)

To be able to profit of the plugins capability you will need to start Burp with

java -classpath /path/to/xercesImpl.jar:/path/to/burp.jar burp.StartBurp

More information can be found on https://github.com/luh2/PDFMetadata
